# Documentation

This directory contains the documentation for the Fake News Detector project.

## Overview

The Fake News Detector is a full-stack web application that uses machine learning to classify news articles as real or fake. It features a Next.js frontend, FastAPI backend, TF-IDF + Logistic Regression model, and SQLite persistence with retraining capabilities.
